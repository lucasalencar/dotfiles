# Global OpenCode Instructions

## tmux MCP

This machine has MCP server for tmux configured. Use it to read tmux panes when needed.

When I mention a "panel" or "painel", I may be referring to a tmux pane. Use the tmux MCP to find and read it.

### Default behavior

When I don't specify which tmux session or window:
- Default to the **current session** and **current window**

When I reference a specific tmux **window**, search for that window by name or index.

When I reference a specific tmux **session**, search within that session and its windows.
